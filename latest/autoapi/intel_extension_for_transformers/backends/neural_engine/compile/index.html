intel_extension_for_transformers.backends.neural_engine.compile.compile(model, config=None)

The compile interface.

Firstly, use model loader to get the computation graph with corresponding framework. The graph contains nodes and edges, the node is op and the edge is the tensor. Then extract the ops in the graph and pack them to our form. Next exploit these above ops to consist sub-graph, which can see as “a new big op”, like LayerNorm.

Note

There may have different computation flow in one subgraph.

Finally, convert them to .yaml file and .bin file for model configuration and inference.
