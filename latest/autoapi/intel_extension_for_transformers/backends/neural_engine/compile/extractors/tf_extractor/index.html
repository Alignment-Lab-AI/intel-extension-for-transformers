intel_extension_for_transformers.backends.neural_engine.compile.extractors.tf_extractor

The neural engine tensorflow extractor file.

Extract all nodes in the input tensorflow model and convert them to engine python operators. All these python operators will compose the engine graph in the order of the original tensorflow calculation graph. Please noticed that the mapping between operators is not one-to-one correspondence. For more related deatils, please refer to ‘ops’ APIs.

The tensorflowExtractor Class.
