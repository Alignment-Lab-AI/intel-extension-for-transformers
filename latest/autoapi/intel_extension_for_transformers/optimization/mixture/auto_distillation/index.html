class intel_extension_for_transformers.optimization.mixture.auto_distillation.AutoDistillation(model_builder, conf_fname_or_obj, framework='pytorch')[source]

The framework class is designed for handling the whole pipeline of AutoDistillation.

AutoDistillation is composed of three major stages, i.e. Model Exploration, Flash Distillation, and Evaluation. In Model Exploration, a search engine will search for a better compressed model from the architecture design space in each iteration. Flash Distillation is the stage for training the searched model to discover its potential. In Evaluation stage, the trained model will be evaluated to measure its performances (e.g. the prediction accuracy, the hardware performance etc.) in order to select the best model architecture.
