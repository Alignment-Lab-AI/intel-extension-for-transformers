Optimization docs: quantization.html, pruning.html, autodistillation.html, distillation.html,

Docs of other functionalities

These utils may help you: data_augmentation.html does NLP datasets augmentation. benchmark.html is used to measure the model performance. metrics.html defines which metric will be used to measure the model. To measure the status of the tuning model, you can refer to objectives.html. pipeline.html makes it simple to use any model from the Hub for inference on tasks.
